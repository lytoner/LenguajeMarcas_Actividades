6. Formularios, Campos de Entrada y Formspree

El elemento <form> es la etiqueta principal para la recopilación de datos. El atributo action define la URL de envío (en este caso, **Formspree**). Los campos de entrada se crean con <input>, especificando el type (ej. email, radio, submit). El atributo name es esencial, ya que es la clave que usa Formspree para identificar el dato enviado.

Ejemplo de Código:


    <form method="POST" action="[ENDPOINT DE FORMSPREE]">
        <label for="nombre">Nombre:</label>
        <input type="text" id="nombre" name="nombre_usuario" required>
        
        <label for="asunto">Asunto:</label>
        <select id="asunto" name="asunto_consulta">
            <option>Consulta</option>
        </select>
        
        <input type="submit" value="Enviar Datos">
    </form>
